## 專案介紹

此專案為英國十字戲的視窗遊戲，使用C#語言撰寫，並使用.NET Framework 4.8作為開發環境。

專案是在南臺科大108學年《物件導向程式設計》課程中，以小組合作期末完成作品。

功能應該都有，只是年久可能失修，或許有些BUG存在，但基本上是可以玩的。

現在放上Github，或許改天看到可以拿出來練練手 :thinking:
